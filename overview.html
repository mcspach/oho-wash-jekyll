Layouts

All layouts have been removed except for default and empty. Front Matter now controls column and wrapping classes, which should allow for easier integration. See the basic-page.html front matter for a working example.

Front Matter Options:

  
    --
    layout: default # This should not change without good reason.
    title: Page Title

    # Set the (Drupal) content type machine name
    content_type: page|home|landing_page|etc.

    # Activate OHO Alert
    oho_alert: true

    # Add Breadcrumbs. Current page added automatically.
    breadcrumbs:
      - Home
      - First Level
      - Second Level

    # Add page-specific CSS and JS.
    css:
      - /path/to/css.css
    js:
      - /path/to/js.js

    # Add components to include before <div class="main-content">
    content_top:
      - component/to-include.html

    # Add components to include after <div class="main-content">
    content_bottom:
      - component/to-include.html

    # Set any additional classes on <div class="page">
    page_classes: class-to-add

    # Set wrapper classes, on the first <div> inside <div class="main-content">
    wrapper_classes: row

    # Set additional classes on <div class="content">
    content_classes: columns medium-9

    # Include some sidebar content
    sidebar_content: component/to-include.html

    # For sidebar nav include, set content style
    sidebar_content_style: padding-levels

    # Set sidebar classes
    sidebar_classes: columns medium-3

    # Enable Drupal 8 Admin markup
    drupa_admin: true

Adding CSS & JS

CSS & JS are added through the _data/assets.yml file. All sitewide scripts and styles should be added here.
